# snowball
Snowball tracker thrown total, on you and by you!

Important info: This addon works only with SuperWoW!!!

- /snowball reset - Reset counters
- /snowball show - Show tracker
- /snowball hide - Hide tracker
- /snowball save - Manually save statistics
